##How To Use
Clone or download the repository,
``bash
git clone https://github.com/piotrowskaannamilena-blip/react-my-first-ever-portfolio-which-is-beautiful.git

Install dependencies:
react-icons is required for social and tech icons:

``bash
npm install react-icons

npm install react-bits

npm install react-icons gsap

DotGrid background needs GSAP (and its plugin) to be installed. Please run:
`` bash
npm install gsap


npm run dev

Open the app in your browser (default: http://localhost:5173/ if using Vite),

Navigate through the portfolio:
Use the header navigation links (desktop/contact),
Toggle between light and dark themes to see accent color changes.
